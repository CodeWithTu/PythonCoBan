## I. Nhắc lại bài học

>Để có thể làm bài tập phía dưới, mọi người ghi nhớ lại bài đã học
>
>- Lệnh in ra màn hình : <b>print("Nội dung cần in")</b>
>- Lệnh in xuống dòng : <b>print('''Nội dung cần in, xuống dòng''')</b>
>- Lệnh in phép tính : <b>print("2 + 3 =", 2 + 3)</b>
>
